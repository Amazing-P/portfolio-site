# Precious Azubuike Portfolio

Static portfolio website for Precious Azubuike, focused on AI engineering, generative AI, automation, and computer vision projects.

Open `index.html` in a browser or publish the repository with GitHub Pages.

## Visitor assistant email setup

The portfolio includes a visitor assistant in `visitor-assistant.js`. By default it collects lead details and opens the visitor's email app with a prepared message to `precious.azubuike032@gmail.com`.

To send emails automatically from GitHub Pages, create an endpoint with a form/email service such as Formspree, EmailJS, Netlify Forms, or your own API. Then paste the endpoint URL into the `LEAD_ENDPOINT` value in `visitor-assistant.js`.
